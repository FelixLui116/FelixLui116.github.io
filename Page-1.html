Galaga game 
(Java)

The game story is about the spaceship fight to the alien from other spaces. The alien are the invaders,
they try to conquer the Earth. The player is the defender to fight with the alien. The game need player
controls the ship and kill the alien to defend the Earth. In the later part of the game, it will have many
enemies to attack the player. The phone need to use 5.0” size screen or bigger screen, it can show all
the game. In the main menu has two button PLAY and EXIT to chick. The play button will go to the main
Activity to play the game. The exit button will the Exit the game

3D Modelling
(Maya)

In Maya Hierarchies, I put every polygon into a main body. The main
body is the highest hierarchy in the minion model. Inside the main
body, it has been classified as left/ right arm, clothe, glasses, eyes, and
feet. Those part are grouped in one and they are under the main body.
If the main body moves or rotates, then every polygon will do the
same. Also, the other part of the polygon in the hierarchies can be
easily to move and rotate

Non Hero!

Player control: In the beginning, the team designed the character to have only 15 seconds wait
time till he moves and the player could not control the character. After the discussion the team added
the mouse click feature when User clicks on the player, he will move automatically by himself
without waiting for the time limit. The character moves one way only to the next level.

Final Year Project
(Unity2D C#)

This project is using Unity to create an RPG game in the PC platform. The game is about the tank to save
the world. The name of the game is “Tank hero”. The player can control the tank to kill the enemy tank.
The enemy will find the player and attack. The player needs to go through all the level. In this project,
using Unity to create the RPG game. The game coding part is using C# and some Unity online resources
to create and build the game system. In addition, Unity can make a simple physical in the game. The
project chooses Unity, because Unity is the game engine and it is easy for the beginner to learn and can
create the game in a short time. In contrast to the C++, SFML and android studio, Unity can simulations
multiple platforms. This project is Use unity 5.4.0f3 (64-bit) version to run.
